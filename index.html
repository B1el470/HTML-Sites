Tema 1: oq é TAGS?

As TAGS são elementos que definem a estrutura e o significado de um documento HTML.

Para que são usadas as TAGS?

Sim, é muito bom e útil saber sobre tags, pois elas funcionam como etiquetas descritivas que ajudam a
organizar, classificar e encontrar informações, seja na internet, na programação ouna organização de dados.
No entanto, a sua importância varia dependendo do contexto.

É bom saber sobre TAGS?

Sim, porque são fundamentais para a construção de páginas web.


Como se deve ser usada uma TAG?

As tags devem ser usadas corretamente para definir a estrutura e o significado do conteúdo.
